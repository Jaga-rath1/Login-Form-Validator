# 🔐 Login Form Validator

A simple and clean login form with email and password validation using **HTML, CSS, and JavaScript**.

---

## 🚀 Features

- ✅ Email validation using Regex
- ✅ Strong password validation
- ✅ Error messages display below inputs
- ✅ Clean and responsive UI
- ✅ Prevents form submission on invalid input

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

## 📌 Validation Rules

### Email:
- Must contain `@` and `.`
- No spaces allowed

### Password:
- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one special character
